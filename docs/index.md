# AWS SAA-C03 备考手册

## 考试概览

AWS Certified Solutions Architect - Associate（SAA-C03）面向具备 **至少 1 年** AWS 实际架构设计经验的工程师，考核基于 AWS Well-Architected Framework 设计解决方案的能力。

| 项目 | 内容 |
|------|------|
| 考试代码 | SAA-C03 |
| 题目数量 | 65 题（含 15 道不计分题） |
| 计分题 | 50 题 |
| 题型 | 单选题 / 多选题 |
| 满分 | 1000 分 |
| 及格线 | **720 分** |
| 考试时长 | 130 分钟 |

> **不计分题说明**：15 道不计分题用于 AWS 收集数据评估未来题目，考试中无法区分哪些是不计分题，正常作答即可。

## 四大考试域与权重

| 考试域 | 内容 | 权重 |
|--------|------|------|
| Domain 1 | 设计安全架构 | **30%** |
| Domain 2 | 设计弹性架构 | **26%** |
| Domain 3 | 设计高性能架构 | **24%** |
| Domain 4 | 设计成本优化架构 | **20%** |

采用**补偿性评分模型**——无需每个域都单独及格，整体达到 720 分即通过。

---

## AWS Well-Architected Framework 六大支柱

这是贯穿整个考试的核心框架，所有架构设计题目都围绕这六个支柱展开：

**卓越运营（Operational Excellence）** 关注运行和监控系统以交付业务价值。**安全性（Security）** 保护信息、系统和资产。**可靠性（Reliability）** 确保工作负载按预期执行。**性能效率（Performance Efficiency）** 高效使用计算资源。**成本优化（Cost Optimization）** 以最低成本交付业务价值。**可持续性（Sustainability）** 最小化环境影响。

> SAA-C03 考试域直接对应其中四大支柱：安全性、可靠性、性能效率、成本优化。

---

## 备考策略

**权重优先**：Domain 1（安全）和 Domain 2（弹性）合计占 56% 权重，是备考重点。

**服务熟悉度**：考试侧重判断"何时用哪个服务"，而非记忆服务参数细节。重点掌握每个服务的**典型使用场景**和**与其他服务的对比区别**。

**场景化思维**：题目通常给出业务场景，要求从多个选项中挑最优解。高频考点包括：最小权限原则、多 AZ 部署、Auto Scaling、S3 存储类别选择等。

**不猜不蒙**：错误选项不扣分，所有题目务必作答。
